# Foundations

Foundation pages extracted from the DS docs site. One file per accepted+crawled URL; see references/foundation-extraction.md for the per-URL iteration contract.

- [colors](./colors.md) — Functional color tokens: surface/foreground/border pairing, mode-awareness, contrast minimums, and semantic role mapping for Primer's `bgColor-*` / `fgColor-*` / `borderColor-*` families.
- [icons](./icons.md) — Icon source package, the `size` enum, and the decorative-vs-contentful accessibility contract for octicons.
- [spacing-layout](./spacing-layout.md) — Spacing scale tokens, the page-region model (header/content/pane/footer), and breakpoint/viewport-range identifiers.
- [responsive](./responsive.md) — User-preference media features, minimum touch-target sizes, and viewport minimums the UI must honor.
- [typography](./typography.md) — Typography tokens: weight/line-height binding, the shorthand `font` tokens, and the decoupling of visual style from semantic heading order.
